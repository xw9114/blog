## 技能概述

SSH（Secure Shell）是现代嵌入式开发与服务器运维的基石。作为一种加密网络传输协议，它为开发者提供了跨越空间的“隐形串口”。本人深度掌握 SSH 在 Client/Server 架构下的应用，能够熟练通过该协议实现对 MaixCAM 等 Linux 设备的远程指令执行、环境配置及代码部署，在无显示终端的场景下确保通信的绝对安全性与操作的高效性。

## 核心能力矩阵

- **加密远程访问与终端控制**：利用 SSH 协议替代物理串口，实现对远端 Linux 系统的 Shell 访问。在复杂的 Wi-Fi 或局域网环境下，通过 **RSA/ED25519 秘钥对**或密码认证方式，安全地进行系统级调试。
- **高效文件传输 (SFTP/SCP)**：基于 SSH 层的安全文件传输协议，实现模型文件（如 YOLO）、源代码与配置文件的无线传输，支持断点续传与目录同步，极大地优化了开发反馈循环。
- **环境隔离与远程开发流**：熟练运用 **VS Code Remote-SSH** 插件，将本地 IDE 的强大编辑能力映射至嵌入式目标机，实现“本地编写、远程执行”的丝滑开发体验，有效隔离开发环境与运行环境。
- **系统自启动与后端驻留**：结合 `rc.local` 或 `systemd`，通过 SSH 远程配置服务的自启动逻辑，并利用 `nohup` 或 `screen` 实现进程在 SSH 断开后的持续运行。

## 代码能力展现

以下展示了如何通过 SSH 命令行实现快速连接、文件拷贝以及利用 Python 库进行自动化远程操作的典型实践：

```bash
# 1. 基础连接：以 root 权限访问指定 IP 的嵌入式设备
ssh root@192.168.1.100

# 2. 安全拷贝：将本地训练好的 YOLO 模型推送到设备指定目录
scp ./yolov8_n.param root@192.168.1.100:/root/models/

# 3. 远程执行：无需登录交互，直接触发设备上的 Python 脚本
ssh root@192.168.1.100 "python3 /root/app/main.py --mode inference"
