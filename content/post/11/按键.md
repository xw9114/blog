## 技能概述

在嵌入式系统开发中，按键交互的稳定性直接决定了产品的工程质量。本技能方案摒弃了传统的阻塞式延时消抖，采用基于 10ms 定时器扫描与状态机判定相结合的核心算法。通过将物理层硬件扫描与逻辑层业务执行彻底解耦，实现了“按下触发”的高实时性与“松手触发”的强鲁棒性，能够轻松应对工业现场的电磁噪声干扰，并为长短按、双击等复杂 HMI 交互逻辑奠定了坚实的底层基础。

## 核心能力矩阵

- **检测与执行分离架构**：底层扫描函数仅负责物理电平检测与时间累计，顶层控制函数全权负责逻辑响应，确保系统任务调度的灵活性与非阻塞性。
- **瞬时状态锁机制**：在按下触发模型中，利用状态计数器的特定瞬时阈值（如 `key_state == 2`）作为单次脉冲触发锁，天然免疫按键重入与机械抖动误触发。
- **闭环信号销毁技术**：在松手触发模型中引入专属的“发射信号值”（如 `254`），强制业务层捕获事件后必须手动重置状态，形成安全闭环，保障状态机的唯一性。
- **多维交互时序扩展**：基于松手瞬间保留的 `key_state` 遗留计数值，可通过时间阈值判定无缝扩展出长按判别及双击沙漏计时等高级人机交互特性。

## 代码能力展现

```c
/* ==========================================
 * 方案一：按下触发 (Press Trigger) - 极速响应
 * 适用场景：急停、开火键、机械臂动作驱动
 * ========================================== */

// 1. 底层扫描 (放入10ms定时器)
void key_scan_press() {
    unsigned char h, l;
    P3 = 0x0f; P4 = 0x00;
    
    if (~P3 & 0x0f) { // 物理按下
        if (key_state < 250) key_state++; 
        
        // 达到消抖时间(20ms)瞬间，产生单次触发锁
        if (key_state == 2) {
            if(P32==0) h=1; else if(P33==0) h=0;
            P3=~0x0f; P4=~0x00;
            if(P35==0) l=2; else if(P42==0) l=1; else if(P44==0) l=0;
            key_value = key_buffer[h][l]; 
        }
    } else { // 物理松开
        key_state = 0; // 立即解锁
        key_value = 0;
    }
}

// 2. 顶层逻辑 (主循环调用)
void key_con_press() {
    if (key_state == 2) { // 捕捉瞬时锁
        switch (key_value) {
            case 9: Emergency_Stop(); break; // 按下瞬间立即生效
        }
    }
}

/* ==========================================
 * 方案二：松手触发 (Release Trigger) - 逻辑严密
 * 适用场景：菜单切换、参数保存、复杂逻辑判断
 * ========================================== */

// 1. 底层扫描 (放入10ms定时器)
void key_scan_release() {
    unsigned char h, l;
    P3 = 0x0f; P4 = 0x00;
    
    if (~P3 & 0x0f) { // 按下期间攒时间
        if (key_state < 250) key_state++; 
        if (key_state == 2) {
            if(P32==0) h=1; else if(P33==0) h=0;
            P3=~0x0f; P4=~0x00;
            if(P35==0) l=2; else if(P42==0) l=1; else if(P44==0) l=0;
            key_value = key_buffer[h][l];
        }
    } else { // 松开瞬间判定合法性
        if (key_state >= 4 && key_state < 250) {
            key_state = 254; // 发射合法松手信号
        } else if (key_state != 254) {
            key_state = 0; // 过滤杂波抖动
        }
    }
}

// 2. 顶层逻辑 (主循环调用)
void key_con_release() {
    if (key_state == 254) { // 捕捉合法松手信号
        key_state = 0; // 【核心安全机制】：必须手动销毁信号
        
        switch (key_value) {
            case 4: jm_state = (jm_state + 1) % 5; break; // 界面丝滑切换
        }
    }
}
